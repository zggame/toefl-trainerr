# TOEFL Speaking Research — Task 1: Test Structure & Task Types

**Last updated:** 2026-04-20

**Critical note:** ETS made significant changes to the TOEFL iBT in July 2023, and changed the scoring scale in January 2026. The information below reflects the current test format.

---

## 1. Current Test Structure (2023–2026)

The TOEFL iBT is now a shortened test (~2 hours total) with four sections:

| Section | Tasks | Time |
|---------|-------|------|
| Reading | 50 items (Complete the Words, Read in Daily Life, Read an Academic Passage) | 30 min |
| Listening | 47 items (Listen and Choose a Response, Listen to a Conversation, Listen to an Announcement, Listen to an Academic Talk) | 29 min |
| Writing | 12 items (Build a Sentence, Write an Email, Write for an Academic Discussion) | 23 min |
| Speaking | 11 tasks (Listen and Repeat, Take an Interview) | 8 min |

**Source:** [ETS TOEFL iBT Test Content](https://www.ets.org/toefl/test-takers/ibt/about/content)

### Speaking Section Details

The speaking section now consists of **11 tasks** in approximately **8 minutes**, split into two types:
- **Listen and Repeat** — short listening + spoken repetition
- **Take an Interview** — respond to interview-style questions

This is a **significant departure** from the old format which had 4 tasks (1 independent + 3 integrated) in 16 minutes.

**Comparison — Old vs. New Speaking Format:**

| Aspect | Old Format (pre-2023) | New Format (2023+) |
|--------|----------------------|-------------------|
| Tasks | 4 | 11 |
| Time | ~16 min | ~8 min |
| Task 1 | Independent — speak about familiar topic (15s prep, 45s response) | Listen and Repeat |
| Task 2 | Integrated Reading/Listening/Speaking (30s prep, 60s response) | Take an Interview |
| Task 3 | Integrated Listening/Speaking — campus (20s prep, 60s response) | (interview tasks) |
| Task 4 | Integrated Listening/Speaking — academic (20s prep, 60s response) | (interview tasks) |

## 2. Scoring Scale (Updated January 2026)

As of January 21, 2026, TOEFL iBT scores use a **1–6 scale** (half-point increments), aligned to the CEFR:

| Score | Speaking (0–30 equivalent) | CEFR Level |
|-------|---------------------------|------------|
| 6 | 28–30 | C2 |
| 5.5 | 27 | — |
| 5 | 25–26 | C1 |
| 4.5 | 23–24 | — |
| 4 | 20–22 | B2 |
| 3.5 | 18–19 | — |
| 3 | 16–17 | B1 |
| 2.5 | 13–15 | — |
| 2 | 10–12 | A2 |
| 1.5 | 5–9 | — |
| 1 | 0–4 | A1 |

For a two-year transition period, a comparable 0–120 scale is also provided.

## 3. Old TOEFL Speaking Task Structure (for reference — still useful for practice content)

Before the 2023 revision, the TOEFL Speaking section had 4 tasks. Even though the current test uses a different format, these task types are still excellent practice material for developing speaking skills. Many third-party prep materials and practice platforms use this format.

### Task 1: Independent Speaking — Personal Experience/Opinion

- **Preparation time:** 15 seconds
- **Response time:** 45 seconds
- **Prompt style:** Describe something or state an opinion (e.g., describe a place, choose between two options, talk about a person)
- **Example prompts:**
  - Describe a place you enjoy visiting in your free time. Explain why this place is important to you.
  - Some people prefer to live in large cities, while others prefer rural areas. Which do you prefer and why?
  - If you could have any job, which would you choose and why?

### Task 2: Integrated Reading/Listening/Speaking — Campus Situation

- **Preparation time:** 30 seconds
- **Response time:** 60 seconds
- **Format:** Read a short passage (~75 words), listen to a conversation about the same topic, then respond by synthesizing both
- **Example:** Reading announces a policy change; conversation discusses student opinions. Response explains the change and the speaker's opinion about it.

### Task 3: Integrated Listening/Speaking — Academic (Reading involved optionally in older version)

- **Preparation time:** 20 seconds
- **Response time:** 60 seconds
- **Format:** Listen to an academic lecture segment, then summarize and explain key points
- **Example:** Professor lectures on a concept (e.g., how bees communicate). Response summarizes the concept and examples.

### Task 4: Integrated Listening/Speaking — Academic (Short lecture)

- **Preparation time:** 20 seconds
- **Response time:** 60 seconds
- **Format:** Listen to a short academic lecture (~2 min), then explain the main idea and supporting details
